# 机器学习基础

## 什么是机器学习

机器学习是人工智能的一个子领域，它使计算机系统能够从数据中学习，而不需要被明确编程。机器学习的核心思想是：通过经验（数据）自动改进性能。

## 学习类型

### 监督学习 (Supervised Learning)
- 使用标注数据进行训练
- 典型任务：分类、回归
- 常见算法：线性回归、决策树、支持向量机、神经网络

### 无监督学习 (Unsupervised Learning)
- 使用未标注数据
- 典型任务：聚类、降维、异常检测
- 常见算法：K-Means、PCA、自编码器

### 强化学习 (Reinforcement Learning)
- 通过与环境交互获取奖励信号进行学习
- 典型应用：游戏AI、机器人控制、推荐系统
- 核心概念：状态、动作、奖励、策略

## 模型优化

### 过拟合与欠拟合
- **过拟合**：模型在训练集上表现好，但泛化能力差
- **欠拟合**：模型过于简单，无法捕捉数据中的模式
- 解决方案：正则化、交叉验证、数据增强、早停

### 超参数调优
常用方法：
1. 网格搜索 (Grid Search)
2. 随机搜索 (Random Search)
3. 贝叶斯优化 (Bayesian Optimization)
4. 自动机器学习 (AutoML)

### 评估指标
- 分类：准确率、精确率、召回率、F1分数、AUC-ROC
- 回归：MSE、RMSE、MAE、R²
- 模型选择需要根据具体业务场景选择合适的指标

## 深度学习

深度学习是机器学习的子集，使用多层神经网络：

- **CNN** (卷积神经网络)：图像处理
- **RNN/LSTM** (循环神经网络)：序列数据
- **Transformer**：自然语言处理的革命性架构
- **GAN** (生成对抗网络)：数据生成

Transformer 架构是当前大语言模型 (LLM) 的基础，如 GPT、BERT、Gemini 等。
